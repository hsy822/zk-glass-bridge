# Glass Bridge ZK Game

A Squid Game-style glass bridge game with Zero-Knowledge proof integration.  
Built with Three.js and Noir to generate proofs directly in the browser.

## Usage

- **Start dev server**  
  ```bash
  bunx vite
  ```

- **Install packages**
  ```bash
  bun add <package-name>
  ```

- **Build project**  
  ```bash
  bunx vite
  ```
